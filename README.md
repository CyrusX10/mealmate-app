# 🍽️ MealMate – Smart Food Waste Reduction App

> A web-based application that helps households reduce food waste through intelligent inventory management, expiry alerts, donation facilitation, and responsible meal planning.

---

## 📌 Project Overview

MealMate is developed as part of the **BIT216 – Software Engineering Principles** assignment at **HELP University / Mid-Valley International College (MVIC)**, Semester May 2026.

The app addresses the growing problem of household food waste by giving users the tools to:
- Track what food they have and when it expires
- Plan weekly meals around available ingredients
- Donate surplus food to the community
- Monitor their food-saving impact through analytics

## 🚀 Features

- **User Registration & Privacy Settings** — Secure sign-up with 2FA and configurable privacy controls
- **Food Inventory Management** — Log food items with expiry dates, quantity, and category; convert items to donations
- **Browse & Claim Donations** — Browse available food listings by location or type and claim items
- **Food Analytics Dashboard** — Visual reports on food saved and donations made
- **Smart Notifications** — Alerts for expiry dates, claimed donations, and meal planning reminders
- **Weekly Meal Planner** — Plan meals based on available inventory with recipe suggestions

## 🤝 Git Workflow

We follow a **feature-branch workflow**:

1. `main` — stable, production-ready code only
2. `dev` — integration branch for ongoing development
3. `feature/[feature-name]` — individual feature branches (e.g. `feature/user-registration`)

Each feature branch is reviewed via a **Pull Request** before merging into `dev`.
